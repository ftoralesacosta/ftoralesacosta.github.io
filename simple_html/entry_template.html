Diffusion for Complete Generation of Collider Events

Machine Learning • Physics • 2024

This project focuses on a deep learning approach designed to simulate high-energy collider events entirely through generative models. Traditional Monte Carlo setups are computationally expensive, so pushing the boundaries using diffusions offers an exciting alternative.

The Problem

In high energy physics, understanding the fundamental particles requires analyzing billions of collision events. Simulating these events is traditionally done using a chain of complex physical models, which takes an enormous amount of computational time.

Key Insight: By treating the distribution of particles as a generation task, we can use diffusion models to drastically reduce the time it takes to create highly accurate mock data.

Methodology

We implemented a diffusion model capable of handling the highly varied and structured data outputted by these simulated collisions. The architecture needed to address several key challenges:

  • Handling varying numbers of particles per event.
  • Respecting the physical conservation laws (like energy and momentum).
  • Generating the fine-grained details of the particle jets.

Results

The resulting model successfully generated events that are statistically indistinguishable from traditional Monte Carlo generators on several key metrics, operating at a fraction of the computational cost.
